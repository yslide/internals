fn emit(&self, form: EmitFormat, config: EmitConfig) -> String

Emit self with the given [EmitFormat][EmitFormat].

NB: This is a multiplexer of the corresponding emit_ methods present on [Emit][Emit], except for [EmitFormat::Latex][EmitFormat::Latex], which is emitted via [emit_wrapped_latex][Emit::emit_wrapped_latex].
